Quick Start

Get Running in Minutes

Clone, install, and launch — GATI is designed to be developer-friendly from day one.

1

Clone & set up environment

Create a virtual environment and install Python dependencies.

git clone https://github.com/your-org/Google-solution
cd Google-solution
python3 -m venv .venv && source .venv/bin/activate
pip install -r backend/requirements.txt
2

Configure environment variables

Add your OpenAI key to enable the full GPT-4o Neural Engine. Without it, the professional simulation engine activates automatically.

cp backend/.env.example backend/.env
# Edit backend/.env → set OPENAI_API_KEY=sk-...
3

Start the backend API

Launch the FastAPI server on port 8000.

uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
4

Start the frontend

Install Node dependencies and launch the Vite dev server on port 5173.

npm run frontend:install
npm run dev
5

(Optional) Run the telemetry simulator

Populate the system with live simulated fleet data.

python -m backend.simulation.simulator
